Event-Driven Python ETL Project

2020

This is a project I finished for an ACloudGuru challenge during the month of September 2020. Utilizing CloudWatch, Lambda, DynamoDB, SNS, and QuickSight, I built an ETL Pipeline to pull data from a public New York Times GitHub repo that tracked the amount of COVID deaths and new cases daily, as well as pulling just the amount of daily recoveries from a daily John Hopkins repo. The pipeline then combines the data and loads it into a DynamoDB table, where it can be exported daily through AWS Data Pipeline to be visualized in AWS QuickSight. All of this is automated and functional, and has an AWS SAM template anyone can deploy to get started. All of this is located in my GitHub. Read more in depth about my approach to the challenge on my blog here.
